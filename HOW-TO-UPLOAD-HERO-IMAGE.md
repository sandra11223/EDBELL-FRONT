## What This Does

- Automatically replaces `/public/hero-professional.jpg`
- Updates the hero section image on the home page
- Maintains all animations and effects
- Works with any image format (JPG, PNG, etc.)

## Recommended Image Specs

- **Format**: JPG or PNG
- **Size**: At least 800x600 pixels
- **Aspect Ratio**: 4:3 or 16:9 works best
- **File Size**: Under 2MB for fast loading
- **Content**: Professional photo with good lighting

## Troubleshooting

**If the image doesn't appear:**
- Make sure you clicked "Upload Hero Image" button
- Wait for the success message (green box)
- Hard refresh the home page: `Ctrl + Shift + R`
- Check that the image file is a valid format

**If upload fails:**
- Check that the file is an image (JPG, PNG, etc.)
- Make sure the file isn't too large (under 5MB)
- Try a different image format
- Check browser console for errors (F12)
